# Task 3 - Dashboard Design

## Objective
The objective of this task is to design an interactive dashboard that helps business stakeholders analyze sales performance and gain meaningful insights from the data.

## Tools Used
- Power BI
- Microsoft Excel / CSV Dataset

## Dataset
Financials Dataset

## Key Performance Indicators (KPIs)
- Total Sales
- Total Profit
- Gross Sales
- Units Sold

## Visualizations Used
1. Sales by Country (Bar Chart)
2. Profit by Segment (Pie Chart)
3. Monthly Sales Trend (Line Chart)
4. Product-wise Sales (Column Chart)
5. Sales vs Profit (Scatter Chart)
6. KPI Cards for Sales, Profit, Gross Sales, and Units Sold

## Filters/Slicers
- Country
- Segment
- Product
- Year

## Insights
- Sales performance varies across different countries.
- Some business segments contribute more profit than others.
- Monthly sales trends show fluctuations over time.
- Certain products generate higher revenue and profit.
- Interactive filters enable better analysis and decision-making.

## Conclusion
The dashboard provides a clear and interactive view of business performance. It helps stakeholders monitor key metrics, identify trends, and make data-driven decisions effectively.

## Outcome
Successfully created an interactive Power BI dashboard for visualizing and analyzing sales data.
